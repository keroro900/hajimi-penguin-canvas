# 贞贞的无限画布 v2.2.9

本版继续走 Electron 自动更新发布，重点修复导演分镜台、RH 工具箱持久化、上传/输出图像能力轨和 Codex 生图工作台登录链路。

## 主要更新

- 导演分镜台每个分镜新增“复用输入”：可从其他分镜一键应用提示词、图像、视频、音频、帧模式和素材顺序，减少重复配置。
- 首尾帧桥接默认折叠关闭，用户勾选后才展开当前相邻分镜的桥接设置，避免占用分镜编辑空间。
- 首尾帧桥接提示词新增 50 条通用高质量预设，支持套用、保存自定义、导入和导出 JSON，方便多台电脑复用。
- 导演分镜台输出绑定继续加固：分镜视频和桥接视频按独立 key 匹配，避免桥接生成后覆盖 S2 等普通分镜输出。
- 时间线拖拽继续加固：分镜右侧拖拽条和中间桥接分隔把手都标记为 resize handle，画布按钮拦截逻辑不会吞掉拖拽。
- RH 工具箱新增持久化 manifest：制作器保存后的应用会写入本地数据文件，RH 工具箱和图像能力轨会读取并合并，打包前同步进发布 manifest。
- RH 工具箱发布检查和 Electron 后置检查改为按 manifest 动态校验所有启用工具，避免新增工具打包时遗漏。
- 上传素材 / 输出素材的图像能力轨新增扩图和消除主体；扩图提供常用分辨率选择，选择框可点击画布空白关闭，并会把真实 LIST 选项值传给 RH。
- Codex 生图工作台新增和 Codex CLI Agent 一致的登录入口、刷新状态、命令复制和可执行路径配置，不再必须先去另一个节点登录。
- 左下角特别注意事项更新到 2026.06.17，补充 gpt-image-2 的 azure 特价分组、2K/4K 稳定性说明和质量参数说明。
- Electron 安装包继续使用 normal 压缩，保留 ffmpeg sidecar 并排除重复运行时，降低 NSIS 打包失败风险。
