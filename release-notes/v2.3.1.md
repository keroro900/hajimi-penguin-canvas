# 贞贞的无限画布 v2.3.1

本次 Electron 自动更新版是 v2.3.0 牧场物语主题发布后的体验修复包，重点收口用户反馈的面板布局、网页图片反推、扩展 API、历史记录、画板/图像编辑和复制粘贴落点。

## 主要更新

- 修复 ModelScope 网页图片右键反推偏离原图的问题：后端会先下载网页图片并转为 `data:image/...;base64` 再发送给视觉模型，避免模型拿不到外链后凭空生成无关提示词。
- 扩展 API 的 Agnes AI 接入补齐 LLM、图像和视频调用链路，视频支持比例、分辨率、时长、帧率和帧数参数，并强化失败信息展示与轮询重试。
- 快捷栏新增历史记录入口，按图像、视频、音频、文本和 3D 分类查看当前画布生成内容；历史记录只保存轻量索引，避免大素材重复进入画布状态。
- 牧场物语主题继续修复右侧控制台双栏展开、折叠态顶部看板、公告栏位置、画布提示气泡、画板节点高度和快捷工具独立操作等布局问题。
- 牧场物语建筑美术资源与可放置预览继续美化，减少默认几何占位感，更贴近轻量牧场经营画风。
- 图像编辑与画板节点补齐常用形状、实心填充选项、Shift 等比例绘制，修复箭头端点位置；画板节点在牧场主题下底部区域高度保持一致。
- 3D 全景节点比例选项收敛为 `21:9`、`2:1` 和自定义，减少无关比例干扰。
- 设置里新增界面字体相关能力，支持更适合大画布缩放阅读的字体配置。
- 修复 ComfyUI 超市 / 工作流输入推断过度约束的问题，避免强制要求图片或提示词输入，按导入配置执行。
- 复制 / 连边粘贴到鼠标位置时，复制内容包围盒左上角对齐鼠标，不再让节点中心落在鼠标处。
- 跨画布生成完成时，非当前画布会在左侧画布列表显示轻量红点提醒；用户切换到对应画布后自动消失。
- 修复高层弹窗上方仍残留左侧快捷栏搜索按钮的问题，避免浮层被画布悬浮控件穿透。
