# 贞贞的无限画布 v2.3.6

本次 Electron 自动更新版继续收口 VibeX 工作台、网页图片反推插件和批量素材处理体验，让插件安装路径更清楚、批处理状态更可见，模型注意事项也同步到最新说明。

## 主要更新

- VibeX 工作台顶部新增 Chrome 插件安装提示：安装包内插件目录为 `resources/extension/web-image-reverse/`，可在 Chrome 扩展程序页开启开发者模式后加载已解压目录。
- 网页图片反推 / RunningHub / VibeX 桥接 Chrome 插件 manifest 版本升级到 `1.1.0`，Electron 打包继续把插件完整放进安装目录。
- 批量素材处理节点改为一次只能启用一种处理策略，避免同时勾选抠图、扩图、高清放大后用户无法判断实际执行链路。
- 批量素材处理并发执行时，素材队列会显示等待、处理中、完成和失败状态，让用户能看出当前哪些素材正在并发处理。
- 左下角模型注意事项补充 Seedance2.0 mini、原生4K、满血 720P 和 FAL / Seedance 预扣后多退少补说明。
- 顶部画布教程更新到第十三弹，覆盖牧场物语主题、图像编辑、Chrome 插件反推、历史记录、Agnes 多模态等近期能力。

## 验证与发布状态

- 公开版本号同步到 `2.3.6`：`package.json`、`package-lock.json`、`vite.config.ts/js`、`electron/main.cjs`、`backend/src/config.js`、`features.json`、`README.md` 与本 release notes 同步。
- 本次发布 GitHub Release `v2.3.6` 并作为 latest 自动更新源。
- 发布入口继续保留确认闸；只有用户明确要求发布后，才允许设置 `T8_RELEASE_APPROVAL=release-2.3.6` 执行上传。
- Release 资产包含 NSIS 安装包、blockmap 与 `latest.yml`。
- 公开源码继续排除 `skill.md`、`roadmap.md`、`local-private/**`、`hermes-agent/**`、`comfyui-anima-t8/**`、Electron 缓存目录、`artifacts/**`、本地维护者工具源码和任何本地私有业务实现。
