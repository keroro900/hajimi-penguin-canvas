# 贞贞的无限画布 v2.3.2

本次 Electron 自动更新版继续收口 v2.3.1 后的用户反馈，重点覆盖网页图片反推插件、RH 自动抠图、放置栏隐藏、主题折叠避让和牧场物语连线层级。

## 主要更新

- 网页图片反推插件修正发送模式：只发送图片时不再同时生成提示词节点；图文一起发送时提示词写入输出素材节点；只发送提示词仍生成文本节点。
- 网页图片反推浮层新增“生成图片”按钮，用户可先修改反推提示词，再直接按当前提示词调用 T8 后端 ModelScope 生图，并把新图发送回画布。
- ModelScope 网页反推后端固定使用 qwen-web 同款 `Qwen/Qwen3-VL-235B-A22B-Instruct` 视觉模型，不再被请求里的普通聊天模型覆盖；当视觉模型返回“无法读取图片 / 未检测到图像数据”等内容时会停止继续生图。
- 画板节点新增 `RH自动抠图`，选中图片后调用 RH 工具箱高清抠图应用层，结果直接替换同一个图片元素并保持选中。
- 图像编辑弹窗新增 `RH抠图`，普通编辑模式替换当前底图，组合模式替换当前选中的图片图层；抠图后后续裁剪、遮罩、画板和宫格操作都基于新图继续处理。
- 图像编辑里按住 Shift 绘制矩形、圆形、圆角矩形和菱形时按原图自然像素比例锁定，避免竖图上画出来仍是长方形。
- 左下放置栏新增隐藏按钮，左侧控制条新增统一的展开/折叠按钮；清空放置栏后默认隐藏，新加入节点会自动恢复显示。
- 侧边栏收缩时，牧场物语顶部看板 / 快捷栏 / 公告栏和圣斗士顶部栏会避让侧边栏折叠按钮，不再和按钮重叠。
- 牧场物语主题修复 ReactFlow 端口、缩放点和连接线剪裁点层级：入口/出口不会被节点内容吞掉，剪裁点也不会盖到节点内容上。
