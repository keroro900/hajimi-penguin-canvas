# 贞贞的无限画布 v2.3.4

本次 Electron 自动更新版继续收口 Cowart 启发的画布创作工作流，让“标注哪里、改哪里、结果落哪里”形成完整链路，同时增强当前画布资源包的可迁移信息。

## 主要更新

- 图像编辑画板新增“标注改图”：用户在图片上画箭头、框线、圆形、标号或文字后，可直接发送干净原图 + 标注合成图进行 AI 改图。
- 输出素材节点和上传素材节点支持标注改图结果自动回落：默认在当前节点右侧创建干净结果；如果选中生成目标框，则直接填入目标框。
- 标注改图请求会保留结构化元数据：来源节点、干净原图、标注图、说明文字、标注数量、模型信息和最终结果 URL，方便后续追踪。
- 当前画布资源包 manifest 增强：导出时额外带上资源库分类 / 素材索引、缩略图引用和生成历史摘要，导入时可识别这些轻量引用并刷新资源库状态。
- 网页图片反推 Chrome 插件继续纳入公开发布边界，确保右键反推、生成图片和发送回画布链路不漏。
