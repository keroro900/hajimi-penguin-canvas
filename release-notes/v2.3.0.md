# 贞贞的无限画布 v2.3.0

本次 Electron 自动更新版重点发布牧场物语主题的阶段性可玩版本：它不再只是主题换色，而是在整张画布上加入种植、建筑、装饰、订单、反馈和成就摘要。

## 主要更新

- 新增第 13 套强识别主题「牧场物语主题」：草地网格、木牌节点、农具端口、麻绳 / 水渠 / 田埂连线、镰刀剪线、牧场 MiniMap 标记和主题音乐 preset。
- 新增全画布牧场养成层：用户可在 ReactFlow 世界坐标里开垦、播种、浇水、过天、收获、完成订单、建造建筑、放置装饰和移动对象，不限制在小面板内。
- 牧场玩法形成 10 分钟闭环：初始资源、4 种作物、金币 / 种子 / 水量 / 木材 / 石头、每日订单、动物产出、NPC 来访、天气 / 节庆、轻量惊喜事件和长期目标。
- 牧场控制台重排为“上方监控看板 + 右侧可折叠控制台 + 顶部快捷工具”，减少面板堆叠和找按钮成本；右侧栏目默认折叠并持久化用户习惯。
- 顶部快捷工具新增牧场控制台按钮，并把常用操作映射到信息栏下方：选择、锄地、播种、浇水、收获、铲除、建造、装饰、移动、删除。
- 修复牧场主题点击落点漂移，牧场对象使用 ReactFlow 世界坐标和视口转换，画布缩放 / 平移后点击位置与实际落点一致。
- 道路和木栅栏支持自动衔接视觉，单个对象和连续对象显示更自然，减少相邻对象叠得杂乱的问题。
- 缺水、成熟、枯萎、订单、动物、来访、建筑收益等状态支持从面板自动跳转到最近目标，并通过短闪、MiniMap 路线和低遮挡反馈确认位置。
- 牧场 HUD 和反馈系统加强：短反馈、今日小目标、每日总结、连击奖励、奖励印章、建筑收益、稻草人守护、资源预期标签和完成回执都可见可读。
- 浅色 / 深色主题可读性统一加固，顶部监控栏、快捷工具、右侧控制台、按钮、TAB、展开卡片和小字号标签都避免灰字、重叠和裁切。
- 完成真实浏览器截图验收：浅色 / 深色均确认顶部监控看板不和右上快捷栏重叠，顶部快捷工具不被垂直裁切，右侧控制台不越界。
