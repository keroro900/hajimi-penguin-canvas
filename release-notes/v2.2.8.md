# 贞贞的无限画布 v2.2.8

本版继续走 Electron 自动更新发布，重点是修复 v2.2.7 后用户真实使用中反馈的导演分镜台、3D 全景分镜提示板、画板尺寸和素材引用细节。

## 主要更新

- 导演分镜台时间线拖拽更稳：分镜右侧拖拽条和中间 `↔` 分隔把手都可调整时长，ReactFlow 不会抢走拖拽事件，时长继续限制在 4-15 秒。
- 导演分镜台桥接片段改为相邻镜头独立后期任务：主镜头生成不自动提交桥接，S1-S2、S2-S3 等桥接可分别设置、分别生成、互不阻塞，并可重新获取已完成结果。
- 导演分镜台输出更清楚：每个输出节点只显示对应分镜或桥接的提示词和视频，避免所有提示词混在一个视频节点里。
- 导演分镜台支持从资源库导入图像、视频、音频，并修复素材无法拖动排序的问题。
- 通用 `@` 素材提及输入支持紧贴文字触发，例如 `女人@` 可直接弹出素材列表，插入后自动补前置空格。
- 上传素材来自资源库时也能作为图像节点参考图正常识别，避免本地上传可用、资源库拖出丢参考图的差异。
- 3D 全景分镜提示板实时预览改为白字、小字号并跟最终输出一致；场景输入框支持回车换行，预设提示词支持插入、保存、导入和导出。
- 画板节点自由尺寸输入不再被旧下限打断，新增保持原图像素尺寸入口，适合长图、小图和原尺寸编辑。
- 宫格剪裁继续加固单张 2x2 拆分，只输出一组 4 张素材，不再重复生成多个相同输出节点。

## 验证与发布

- 本次公开版本号同步到 `2.2.8`：`package.json`、`package-lock.json`、`vite.config.ts/js`、`electron/main.cjs`、`backend/src/config.js`、`features.json`、`README.md` 与本 release notes 同步。
- Release 资产包含 NSIS 安装包、blockmap 与 `latest.yml`，自动更新仍使用 GitHub Release latest 渠道。
- Electron 包体继续保留 `tools/ffmpeg` sidecar，并排除重复的 `@ffmpeg-installer` 运行时，避免 NSIS 安装包超过 Windows 编译器内存映射上限。
- 公开源码继续排除 `skill.md`、`roadmap.md`、`local-private/**`、`hermes-agent/**`、`comfyui-anima-t8/**`、Electron 缓存目录和本地维护者工具源码。
