# 哈基米画布 v2.4.0

<!-- t8-release:v2.4.0 -->

## 更新方式

- 桌面端仍由用户主动点击“检查更新”；发现新版本后，再由用户主动下载并确认打开安装向导。
- 不会后台自动下载，也不会在退出应用时静默安装。
- Release 提供 `JIMI AI-Setup-2.4.0.exe`、对应 blockmap 与 `latest.yml`。

## 兼容范围

- 已核验的 `2.3.8` 安装包内更新配置均指向 `keroro900/hajimi-penguin-canvas`，可通过应用内更新迁移到 v2.4.0。
- 若用户持有指向其他仓库或没有更新配置的历史安装包，需要先手动安装一次本仓库版本；后续版本即可使用应用内更新。

## 本次修复

- 统一默认服务和扩展平台的图像、视频、LLM 模型配置与协议选择。
- 修复画布交互、智能节点属性面板、媒体预览和下载文件名。
- 保留用户已经保存的真实上游模型名，避免配置列表暂时不可用时静默切换。
