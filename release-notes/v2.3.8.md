# 贞贞的无限画布 v2.3.8

本次 Electron 自动更新版发布轻量「视频剪辑」核心节点，并同步近期创作者工作流修复：画布内可以直接收集多段视频、裁剪排序、套用简单转场/滤镜，再用本地 ffmpeg 合成为一个标准视频素材。

## 主要更新

- 新增核心节点「视频剪辑」：支持上传多个视频、读取上游视频、右键发送到视频剪辑节点，并可跨画布追加到指定视频剪辑节点。
- 视频剪辑节点提供单段预览、缩略图时间线、拖拽排序、入点/出点裁剪、片段拆分、输出预设、一键模板、转场、滤镜、音频保留/静音策略。
- 合成流程改为后端异步任务：前端显示进度，可取消合成，完成后生成 `/files/output/video_edit_*.mp4`，并支持下载或继续连到输出素材节点。
- 批量素材处理的 RH 并发选择扩展到 1-10，并保持一次只启用一种处理策略，降低多策略误操作。
- 顶部画布教程新增第十四弹，补齐 VibeX 联动、Chrome 反推插件增强、目标框、大画布性能、批量素材节点等教程入口。
- 图像模型说明与映射同步 2026.06.25 谷歌香蕉正式模型名，旧 preview 模型名会自动归一到正式模型名。
- Electron 打包校验新增 `videoOps` 后端路由检查，避免安装包缺失视频剪辑合成接口。
