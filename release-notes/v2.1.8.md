# v2.1.8 Source Release

发布日期：2026-06-13

本次为 GitHub 源码版更新，不打包 Electron，不创建或更新 GitHub Release，不发布 `latest.yml` 自动更新资产。

## 更新内容

- 新增并持续打磨俄罗斯方块主题：方块化画布、节点、连线、小地图和右上角 Tetris 小游戏；默认主题音乐加入 Korobeiniki，游戏关卡、障碍、技能和视觉反馈持续增强。
- 新增工具箱「批量素材处理」节点，支持多文件 / 文件夹导入、批量命名归档、去黑边、抠图、扩图、高清放大、格式转换和打开输出文件夹。
- 新增 Codex CLI Agent 创作者节点，支持简约模式、创作台、流式对话、Skill 调用、项目 Skill、模板工坊、产物库、LLM / IMG 双模式、参考素材可视化和输出素材发布。
- 新增 Codex 生图工作台节点，面向 `imagegen` 生图流程，包含任务队列、公共图库、模板 / 片段工具、参数快捷 Chip、批量历史和画布输出衔接。
- 修复 Codex Agent 多轮创作中的产物提取、产物库删除后复活、文本产物保留、自动滚动、复制文本、深色主题可读性和简约模式发布语义。
- Grok Agent 创作台继续修复视频任务轮询、模式按钮高亮、产物库删除 / 清空 / 批量删除、深色主题可读性和图生视频挂起问题。
- 视频节点补齐 `grok-1.5-video-6s / 10s / 15s` 的 Zhenzhen New 通道参数，按 Comfly 节点实现为这三个模型单独传入 `size`，不破坏其他 Grok Video 比例逻辑。
- AI 消除水印能力跟进新版 remove-ai-watermarks 项目，补齐本地 runtime 文档、后端任务路由和前端节点参数。
- 修复素材拖动、上传 / 输出素材预览、画布分类 / 侧边栏主题可读性、Grok / Codex 创作台文字选择复制等多处体验问题。

## 边界说明

- 本次只推 GitHub 源码，不运行 `npm run dist:release`，不打包 Electron，不创建 GitHub Release。
- 公开提交继续排除 `skill.md`、`roadmap.md`、`local-private/**`、`hermes-agent/**`、Electron 缓存目录、维护者本地制作器源码和任何私有商业能力 / 分组令牌实现。
- `SKILL.md` 为本地协同手册，随本机更新记录本次源码版，但不进入公开仓库。
