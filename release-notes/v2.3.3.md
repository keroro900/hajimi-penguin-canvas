# 贞贞的无限画布 v2.3.3

本次 Electron 自动更新版发布 Cowart 启发的轻量画布创作工作流，重点让用户先确定 AI 结果落点，再把网页反推、选区素材或上游节点生成结果送回指定位置。

## 主要更新

- 右上快捷栏新增“生成目标框”：用户可先在画布上摆好结果框，再输入提示词或连接上游文本 / 图像节点，生成结果可替换到框内或保留版本到右侧输出节点。
- 画布右键选区新增“选区生成图像”：把选中的文本、图片和输出素材整理为提示词与参考图，在选区右侧创建图像生成节点，减少手动复制整理。
- 网页图片反推回传增强：当前选中生成目标框时，反推图片优先填入目标框，提示词模式会写入目标框 prompt，不再额外散落节点。
- 新增当前画布资源包清单导出：默认生成轻量 JSON manifest，记录画布、素材引用、缺失文件报告和可复制文件列表，不写入 base64 大文件、API Key 或私有配置。
- 新增标注改图请求契约：保留干净原图和标注合成图，要求模型按标注意图修改，同时移除箭头、框线、标号和编辑痕迹，为后续一键标注改图打好基础。
- 发布范围继续包含网页图片反推 Chrome 插件源码：`extension/manifest.json`、`extension/scripts/background.js`、`extension/scripts/content.js` 和 `extension/styles/content.css` 随公开仓库保留。
